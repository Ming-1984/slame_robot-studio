# 测绘建图（官方发行包放置说明）

此目录用于放置 **测绘建图（官方上位机）** 的 Windows 发行包（`aurora_remote.exe` 及其依赖文件、`media/` 等）。

## 为什么仓库里没有这些文件？
- 官方发行包属于**闭源二进制**，且体积较大；本 GitHub 仓库只托管我们自己的源码与脚本。

## 你需要放哪些文件？
确保该目录下至少包含：
```
aurora_app/
  aurora_remote.exe
  media/
  *.dll
```

如果你拿到的官方包里自带 `VC_redist.x64.exe`，也建议一并放在这里（用于缺依赖时引导安装）。
