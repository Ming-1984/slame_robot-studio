# Aurora 项目总体计划

## 项目概述

Aurora 项目旨在开发一套点云数据提取与处理工具链，帮助用户自动化完成点云提取、墙体增强与建筑元素检测等流程。  
说明：本仓库已移除 CAD/DXF 输出相关功能；真景建图等能力请使用官方上位机：`/home/jetson/ros2_ws/aurora_remote-release-2.1.0-rc2`。

## 项目目标

1. 实现从 STCM 文件中提取高质量点云数据
2. 开发点云数据处理和分析算法
3. 提供易用的操作界面和工作流程
4. 与官方上位机能力联动与集成

## 项目阶段

### 第一阶段：点云数据提取（已完成）

- [x] Aurora SDK 集成与测试
- [x] STCM 文件解析与处理

### 第二阶段：点云数据处理（进行中）

### 第三阶段：系统集成与优化（计划中）

- [ ] 用户界面开发
